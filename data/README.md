# Данные проекта

В этой папке хранятся только демонстрационные или обезличенные данные.

Большие, закрытые и исходные наборы данных не размещаются в репозитории.

- data/sample/ — небольшой пример данных для проверки работы;
- data/raw/ — локальная папка для исходных данных, не отправляется на GitHub.
